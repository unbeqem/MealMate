## Decisions Made

- UUID v4 text PKs chosen for all domain tables. This is a "schemaVersion 1 or never" decision — retrofitting UUID PKs after feature code is written requires migrations touching every FK reference across all tables.
- Optional `QueryExecutor` parameter on AppDatabase constructor. This allows Phase 8 to pass in a `SqliteAsyncDriftConnection` from PowerSync without modifying the AppDatabase class itself.
- `app_database.g.dart` created as a placeholder. The real generated file is produced by `dart run build_runner build` with the Flutter SDK installed. This is the same environment limitation documented in 01-01: Flutter SDK is not in the local PATH; CI handles build verification.
