2. Create `planner_grid.dart` — a ConsumerWidget:
   - Takes `DateTime weekStart`.
   - Watches `ref.watch(mealPlanNotifierProvider(weekStart))`.
   - Layout: `SingleChildScrollView(scrollDirection: Axis.horizontal)` wrapping a `Row` of 7 day columns.
   - Each day column is a `Column` with: a header showing abbreviated day name + date (e.g., "Mon\n3"), then 3 slot cells (breakfast, lunch, dinner).
   - Each slot cell: find the matching MealSlot from the async data (match on dayOfWeek + mealType). If slot has a recipeId, render `MealSlotCard`. If empty/null, render `EmptySlotCard`.
   - Day column width: fixed at `MediaQuery.of(context).size.width * 0.35` (so ~2.8 days visible in portrait, scrollable).
   - Meal type labels ("B", "L", "D" or icons) on the left side as a fixed column before the scrollable row. Use a `Row` with a non-scrolling Column of labels + the horizontal ScrollView.
   - Handle loading state with shimmer or CircularProgressIndicator. Handle error with error text.

3. Create `meal_slot_card.dart` — a ConsumerWidget:
   - Takes `MealSlot slot` and `DateTime weekStart`.
   - Displays: recipe thumbnail image (Image.network with errorBuilder showing Icons.restaurant placeholder), recipe title below/over image. Compact card style consistent with RecipeCard but smaller.
   - Inline action icons: a small replace icon (Icons.swap_horiz or Icons.edit) and remove icon (Icons.close or Icons.delete_outline) positioned at top-right as a Row of IconButtons.
   - Remove icon: calls `ref.read(mealPlanNotifierProvider(weekStart).notifier).clearSlot(slot.id)`.
   - Replace icon: navigates to recipe picker in "select for slot" mode — `context.push('/recipes?selectForSlot=true&day=${slot.dayOfWeek}&meal=${slot.mealType}&week=${weekStart.millisecondsSinceEpoch}')`.
   - Tapping the card body (not icons): navigates to `context.push('/recipes/${slot.recipeId}')` for recipe detail view.
   - Visual: Card with rounded corners, slight elevation, clipped image at top.

4. Create `empty_slot_card.dart` — a StatelessWidget:

   - Renders a dashed border container (use `CustomPaint` with dashed path or a `DottedBorder`-style Container with `BoxDecoration(border: Border.all(style: BorderStyle.none))` and a custom painter, or simply a Container with a thin grey dashed-looking border using `BoxDecoration` with a dotted pattern).

   - On tap: navigate to recipe picker — `context.push('/recipes?selectForSlot=true&day=$dayOfWeek&meal=$mealType&week=${weekStart.millisecondsSinceEpoch}')`.
