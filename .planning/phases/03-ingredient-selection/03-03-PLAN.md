**1. Create favorites provider (`ingredient_favorites_provider.dart`):**

```dart
@riverpod
class IngredientFavorites extends _$IngredientFavorites {
  @override
  FutureOr<List<Ingredient>> build() async {
    final repo = ref.watch(ingredientRepositoryProvider);
    return repo.getFavorites();
  }

  Future<void> toggleFavorite(String ingredientId) async {
    final repo = ref.read(ingredientRepositoryProvider);

    // Optimistic update: toggle in local state immediately
    state = AsyncData(
      state.valueOrNull?.map((i) {
        if (i.id == ingredientId) {
          return i.copyWith(isFavorite: !i.isFavorite);
        }
        return i;
      }).toList() ?? [],
    );

    // Persist to Drift (syncStatus set to pending for future Supabase sync)
    await repo.toggleFavorite(ingredientId);

    // Refresh from source of truth
    if (!ref.mounted) return;
    state = AsyncData(await repo.getFavorites());
  }
}
```

**2. Create selected-today provider (`selected_today_provider.dart`):**

Follow RESEARCH Pattern 4 and Pattern 5. This is a `@Riverpod(keepAlive: true)` provider — it must survive navigation because Phase 4 recipe discovery reads it.

**CRITICAL CHANGE from original plan:** State is `Map<String, String>` (id -> name), NOT `Set<String>`. The expandable pill bar needs ingredient names for chips (per locked decision: "shows count + first 2-3 ingredient names as chips"). The `toggle()` method signature accepts `name` at toggle time to avoid async lookups.

```dart
@Riverpod(keepAlive: true)
class SelectedTodayNotifier extends _$SelectedTodayNotifier {
  @override
  FutureOr<Map<String, String>> build() async {
    final repo = ref.watch(ingredientRepositoryProvider);
    final user = ref.watch(currentUserProvider);
    if (user == null) return {};
    // getSelectedToday returns ALL IDs for userId — no date filter
    final ids = await repo.getSelectedToday(user.id);
    // We only have IDs from Drift — look up names from cached ingredients
    final db = ref.read(appDatabaseProvider);
    final localSource = IngredientLocalSource(db);
    final map = <String, String>{};
    for (final id in ids) {
      final ingredient = await localSource.getIngredient(id);
      map[id] = ingredient?.name ?? id;
    }
    return map;
  }

  Future<void> toggle(String ingredientId, {required String name}) async {
    final repo = ref.read(ingredientRepositoryProvider);
    final user = ref.read(currentUserProvider);
    if (user == null) return;

    final current = Map<String, String>.from(state.valueOrNull ?? {});

    if (current.containsKey(ingredientId)) {
      current.remove(ingredientId);
      await repo.removeSelectedToday(ingredientId);
    } else {
      current[ingredientId] = name;
      await repo.addSelectedToday(ingredientId, user.id);
    }

    if (!ref.mounted) return;
    state = AsyncData(current);
  }

  /// Batch-add multiple ingredients at once — single state update, avoids N rebuilds.
  /// Per locked decision: "Add all favorites" bulk action.
  Future<void> addAll(List<Ingredient> ingredients) async {
    final repo = ref.read(ingredientRepositoryProvider);
    final user = ref.read(currentUserProvider);
    if (user == null) return;

    final current = Map<String, String>.from(state.valueOrNull ?? {});

    for (final ingredient in ingredients) {
      if (!current.containsKey(ingredient.id)) {
        current[ingredient.id] = ingredient.name;
        await repo.addSelectedToday(ingredient.id, user.id);
      }
    }

    if (!ref.mounted) return;
    state = AsyncData(current);
  }

  Future<void> clearAll() async {
    final repo = ref.read(ingredientRepositoryProvider);
    final user = ref.read(currentUserProvider);
    if (user == null) return;
    // clearSelectedToday removes ALL entries for userId — no date filter
    await repo.clearSelectedToday(user.id);
    if (!ref.mounted) return;
    state = const AsyncData({});
  }

  Set<String> get selectedIds => state.valueOrNull?.keys.toSet() ?? {};
  Map<String, String> get selectedNames => state.valueOrNull ?? {};
  int get count => selectedIds.length;
}
```

**CRITICAL:** Use `@Riverpod(keepAlive: true)` — NOT auto-dispose. This state must persist across screen navigation so Phase 4 recipe discovery can read it from any route. Check `ref.mounted` after every `await` (Riverpod 3 requirement).

**2. Create `SelectedTodayBar` widget — EXPANDABLE pill bar (`selected_today_bar.dart`):**

Per locked decision: "Expandable pill bar at bottom: shows count + first 2-3 ingredient names as chips, tap to expand full list, 'Find Recipes' CTA button when >= 1 selected."

This is a `ConsumerStatefulWidget` using `AnimatedContainer` for expand/collapse (see RESEARCH Pattern 5):

```dart
class SelectedTodayBar extends ConsumerStatefulWidget {
  const SelectedTodayBar({super.key});
  @override
  ConsumerState<SelectedTodayBar> createState() => _SelectedTodayBarState();
}

class _SelectedTodayBarState extends ConsumerState<SelectedTodayBar> {
  bool _expanded = false;

  @override
  Widget build(BuildContext context) {
    final selectedAsync = ref.watch(selectedTodayNotifierProvider);
    final selectedMap = selectedAsync.valueOrNull ?? <String, String>{};
    if (selectedMap.isEmpty) return const SizedBox.shrink();

    final names = selectedMap.values.toList();
    final count = selectedMap.length;

    return AnimatedContainer(
      duration: const Duration(milliseconds: 250),
      curve: Curves.easeInOut,
      // Collapsed: ~64px; Expanded: enough for all chips + CTA
      decoration: BoxDecoration(
        color: Theme.of(context).colorScheme.primaryContainer,
        borderRadius: const BorderRadius.vertical(top: Radius.circular(16)),
      ),
      child: SafeArea(
        top: false,
        child: Column(
          mainAxisSize: MainAxisSize.min,
          children: [
            // Collapsed row: tap to expand, shows count + first 2-3 name chips
            GestureDetector(
              onTap: () => setState(() => _expanded = !_expanded),
              child: Padding(
                padding: const EdgeInsets.symmetric(horizontal: 16, vertical: 8),
                child: Row(
                  children: [
                    Text('$count ingredient${count == 1 ? '' : 's'}',
                        style: Theme.of(context).textTheme.titleSmall),
                    const SizedBox(width: 8),
                    // Show first 2-3 name chips in collapsed state
                    ...names.take(3).map((name) => Padding(
                      padding: const EdgeInsets.only(right: 4),
                      child: Chip(
                        label: Text(name, style: const TextStyle(fontSize: 12)),
                        materialTapTargetSize: MaterialTapTargetSize.shrinkWrap,
                        visualDensity: VisualDensity.compact,
                      ),
                    )),
                    if (count > 3) Text('+${count - 3} more',
                        style: Theme.of(context).textTheme.bodySmall),
                    const Spacer(),
                    Icon(_expanded ? Icons.expand_more : Icons.expand_less),
                  ],
                ),
              ),
            ),
            // Expanded: full chip list + "Clear all" + "Find Recipes" CTA
            if (_expanded) ...[
              Padding(
                padding: const EdgeInsets.symmetric(horizontal: 16),
                child: Wrap(
                  spacing: 4,
                  runSpacing: 4,
                  children: names.map((name) => Chip(
                    label: Text(name),
                    onDeleted: () {
                      final id = selectedMap.entries
                          .firstWhere((e) => e.value == name)
                          .key;
                      ref.read(selectedTodayNotifierProvider.notifier)
                          .toggle(id, name: name);
                    },
                  )).toList(),
                ),
              ),
              Padding(
                padding: const EdgeInsets.all(16),
                child: Row(
                  children: [
                    TextButton(
                      onPressed: () => ref.read(selectedTodayNotifierProvider.notifier).clearAll(),
                      child: const Text('Clear all'),
                    ),
                    const Spacer(),
                    ElevatedButton.icon(
                      onPressed: () => context.push('/recipes'),
                      icon: const Icon(Icons.restaurant),
                      label: const Text('Find Recipes'),
                    ),
                  ],
                ),
              ),
            ] else ...[
              // Collapsed: just the Find Recipes button
              Padding(
                padding: const EdgeInsets.only(left: 16, right: 16, bottom: 8),
                child: SizedBox(
                  width: double.infinity,
                  child: ElevatedButton.icon(
                    onPressed: () => context.push('/recipes'),
                    icon: const Icon(Icons.restaurant),
                    label: const Text('Find Recipes'),
                  ),
                ),
              ),
            ],
          ],
        ),
      ),
    );
  }
}
```
