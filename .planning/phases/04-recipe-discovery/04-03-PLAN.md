    - formatAmount(1.0) returns "1"
    - formatAmount(0.5) returns "0.5"
    - formatAmount(0.6666666) returns "0.67"
    - formatAmount(2.0) returns "2"
    - formatAmount(1.50) returns "1.5" (strips trailing zero)
    - Scaled amount formula: (ingredient.amount / recipe.servings) * selectedServings

    - Scaling 1.5 tsp from 2 servings to 1 serving returns 0.75
